# Fines Management API

Spring Boot REST API for managing fines and persons.

## Tech stack
- Java 21
- Spring Boot 3
- Spring Security (Basic Auth, Role-based)
- Spring Data JPA
- MySQL
- Flyway
- JUnit 5 + Mockito

## Features
- CRUD for Persons & Fines
- Total fine amount per city
- Role-based authorization (ADMIN / USER)
- Fully tested service & controller layers

## Running the project
- Configure MySQL
- Run application
